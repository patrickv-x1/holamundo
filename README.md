Actividad 2.1 - Hola Mundo en JavaScript

Descripción:
Esta actividad consiste en mostrar el mensaje "Hola Mundo" usando JavaScript de diferentes formas.

Archivos:

* actividad_hola_mundo.html
* actividad.js

Ejecución:

1. Abrir el archivo HTML en el navegador
2. Presionar el botón
3. Revisar:

   * Alerta
   * Texto en la página
   * Texto en el div
   * Consola (F12)

Funciones usadas:

* alert()
* console.log()
* document.getElementById()

Observación:
El script se coloca al final del HTML para que funcione correctamente.
